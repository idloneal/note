## 各种符号含义

| 符号       | 含义                   |
| ---------- | ---------------------- |
| +          | public                 |
| -          | private                |
| #          | protected              |
| ~          | package/default        |
| <<抽象类>> | 也可以用斜体表示抽象类 |
| 实线       | 关联                   |
| 箭头       | 继承                   |
| 空心菱形   | 聚合                   |
| 实心菱形   | 组合                   |
|            | 实线上的符号           |
| n          | n个指定对象            |
| 0..1       | 零到一（可不写）       |
| 0..*       | 零到多                 |
| 1..*       | 一到多                 |
| m..n       | 特定的范围             |
