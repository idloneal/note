# 基础

## 命令

| enable                     | 特权模式                   |
| -------------------------- | -------------------------- |
| configure terminal         | 全局配置模式               |
| interface XXX              | 特殊配置模式               |
| host name                  | 改名字                     |
| enable password xxx        | 配置密码为xxx              |
| line vty 0 4               | 配置路由器RTA的虚拟终端0~4 |
| no shutdown                | 开启                       |
| ip address [IP] [子网掩码] | 改ip                       |
