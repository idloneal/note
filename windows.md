# 命令
| dir                                      | 查看当前文件            |
| ---------------------------------------- | ----------------------- |
| chdir                                    | 查看绝对路径            |
| type                                     | 显示文件内容            |
| net user                                 | 显示用户                |
| whoami                                   | 显示当前用户            |
| net user admin 123  /add                 | 添加用户admin密码123    |
| net localgroup Administrators admin /add | 将admin用户添加为管理员 |
